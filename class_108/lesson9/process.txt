我们一般把已经加载在内存中的程序叫做进程，或者一般吧正在运行的程序叫叫作进程

查看进程的操作： ps ajx | grep ...

一个操作系统，不仅仅只能运行一个进程，可以同时运行多个进程，所以操作系统需要把进程管理起来

任何一个进程，加载到内存的时候，需要先创建描述进程（属性）的结构体对象
PCB --- process control block

struct 结构体
{
    进程编号
    进程的状态
    优先级
    ...
}

进程 = 描述进程pcb结构体 + 代码和数据
{
    PCB是由操作系统自己维护的  --  包含描述进程的所有属性值
    代码和数据是由我们维护的  -- 
}
